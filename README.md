# BakaDesktop: Cirno ⑨

**BakaDesktop** — Фумо Чирно из вселенной Touhou Project для вашего рабочего стола.

## Особенности
* **Audio Visualizer:** Синхронизация с музыкой.
* **Memory:** Сохранение состояния.
* **System Tray:** Управление всеми функциями через контекстное меню в трее.
* **Autostart:** Автозапуск при старте ПК.
* **Multi-Language:** Локализация на русский, английский, японский и другие языки.

## ⑨ Чирно?
Чирно — ледяная фея, объявившая себя «Сильнейшей» (да-да!), но в виде Фумо, а по итогу и воплощенная в цифры.
Идею и спрайты взял из плагина fruity dance сделанный для FL studio человеком Resy.
www.youtube.com/@resy1232

##  Проект
Проект написан на **Python 3.13** с использованием следующих библиотек:
* `PyQt5` — графический интерфейс и системный трей.
* `PyAudioWPatch` — захват аудиопотока Windows (WASAPI Loopback).
* `Pygame` — воспроизведение встроенных звуковых эффектов и музыки.
* `Pillow` — обработка графических ресурсов и спрайтов.

Скачать готовый установщик из раздела [Releases](https://github.com/TheMarsHub/BakaDesktop/releases).

2. Запустите инсталлятор и следуйте инструкциям.

### Вариант 2: Сурс код
Если вы хотите запустить проект из исходного кода:
1. Клонируйте репозиторий: `git clone https://github.com/TheMarsHub/BakaDesktop.git`
2. Установите библиотеки.
